# Jupyter Notebook

Install the classic Jupyter Notebook with:
pip install notebook

To run the notebook:
jupyter notebook

---

# JupyterLab

Install JupyterLab with pip:
pip install jupyterlab

Once installed, launch JupyterLab with:
jupyter lab
